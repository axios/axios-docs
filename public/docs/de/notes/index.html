Notes

Semver

Bis Axios eine Version 1.0 erreicht, können breaking changes auch in minor-versionen stattfinden. Zum beispiel sind 0.5.1 und 0.5.4 kompatibel, aber 0.6.0 moglicherweise nicht mehr.

Promises

Axios benötigt das die Promise-API unterstützt wird. Falls ihre Umgebung dieser Vorraussetzung nicht nachkommt können sie diesen Polyfill verwenden.

TypeScript

Axios beinhaltet TypeScript-Definitionen.

import axios from 'axios';
axios.get('/user?ID=12345');
